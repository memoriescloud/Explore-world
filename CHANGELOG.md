# 版本更新记录（CHANGELOG）

> 版本号规则：**双段式 `MAJOR.ITERATION`**
> - **MAJOR**：大版本 / 基础架构 / 破坏性变更（如新增登录、同步协议重构、题库数据结构迁移、框架替换）时 +1，ITERATION 归零。
> - **ITERATION**：功能增强、体验改版、bug 修复等所有非重大更新顺序 +1（不区分「小功能」与「纯修复」）。
> - 线上已有版本（纳入版本管理前）记作 **v1.0 基线**。
>
> 记录规范：
> - 每条更新须含 **更新时间** 与 **上线时间**，均精确到「小时:分」，格式 `YYYY-MM-DD HH:MM`。
> - 不再记录「主要实施方式」。
> - 每次推送云端（Deno Deploy）成功后在对话中列表告知：本次版本号、更新主要内容、上线更新时间。

## v1.1

**更新时间**：2026-07-17 16:40
**上线时间**：2026-07-17 18:23

**更新主要内容**
1. 首页底部「云端同步」板块重构（有框卡片版）：
   - 「云端同步开启」状态胶囊与「设备同步」按钮由两端改为**居中贴近成组**（`justify-content:center; gap:10px`）。
   - 按钮文案「添加设备」→**「设备同步」**（含配对弹窗标题同步修改）。
   - 说明文字「数据保存在本机浏览器，并自动跨设备云端同步。」移入该板块**最下方居中对齐**（带细分隔线），全局 footer 不再重复。
2. 修复「近 7 日练习」图表中**答题量数字**（蓝柱上方数字）被准确率曲线/标签遮挡的问题：
   - 答题量数字改在**最后一层绘制**（始终置顶），并加**白底圆角胶囊**，任何背景下都清晰可读。
   - 与准确率标签同列且纵向重叠时，答题量数字**自动上移错开**，避免相互压盖。
   - 准确率标签（绿）保持白底胶囊，防止压在蓝柱上。
3. **简答题行为调整**：
   - 去掉「我答对了 / 我答错了」自评按钮，改为「显示参考答案 + 下一题」，用户无需判定对错、可直接跳到下一题。
   - 简答题**不计入正确率**：准确率分母改用客观题计题量（`daily.obj`），简答题仅计入「每日答题量」（`daily.count`），不写 `stats` / 错题本。
4. 「近 7 日练习」卡片**高度降低**：图表 viewBox 高 272 → 210（约降 23%）。
5. 间距微调：
   - 近 7 日练习卡片与上方「答题模块」网格间距**加大**（新增 `margin-top:10px`，合计约 26px）。
   - 近 7 日练习卡片与底部同步板块间距**收窄**（`.home-foot` 上内边距 16px → 2px，合计约 10px）。
6. 新增**版本号机制**：`app.js` 加入 `APP_VERSION` 常量；footer 动态显示 `v<版本号>`；本文件记录每次实质性更新。
7. **错题练习体验调整**：
   - 错题练习模式下**移除默认「下一题 →」按钮**，由「保留并继续」承担继续功能：`renderQuizCard` 的 `buildResult` 依据 `opts.skipNext` 决定是否渲染「下一题 →」，`runOneByOne` 传 `skipNext: relax`（仅错题库 relax 模式隐藏）。
   - 「保留并继续」按钮点击后**直接进入下一题**（原本仅提示），样式改为**蓝底白字**（`cls:"btn primary"`）；`runOneByOne` 的 `extraActions` 闭包将 `goNext` 透传给 `extraFor`，按钮 `onClick` 调用 `goNext()`。
   - 错题练习**移除重复限制规则**：`drawId` 新增 `relax` 形参，`relax=true` 时跳过 `isEligible`（即不再限制「同一题每日≤3、20分钟内≤1」），错题库可自由重复练习；`renderWrongPractice` 调用时传 `relax=true`。
8. **副标题规则文字清理**：错题练习、模块练习、整体测试的「第 N 题」副标题统一精简为「<模式> · 第 N 题」格式，移除「（同一题每日≤3、20分钟内≤1）」及整体测试原有的长描述句；随机测试副标题本就不含该文字（「随机测试（不含简答）· 第 X / 10 题」），无需改动。
9. **错题练习改为纯随机抽题**：错题库模式下 `drawId(relax=true)` 不再走加权抽取（近 3 日降权 + 累计出现越少权重越高），改为从错题库题目池 `pool[Math.floor(Math.random()*pool.length)]` **等概率纯随机**抽取，每道错题被抽到概率相同；整体测试 / 模块练习 / 随机测试的加权逻辑保持不变；重复限制（前序已移除）在错题练习仍不生效。
10. **「移出错题本」按钮橙色高亮**：错题练习的「移出错题本」按钮由橙字描边（`btn ghost`）改为**橙底白字**（新增 `style.css` `.btn.warn`：背景 `#ea580c`、白字），与「保留并继续」（蓝底白字）区分；题目详情弹窗内的同款按钮（全局通用）保持原样式不变。

**改动的主要文件**
- `app.js`（同步板块渲染、按钮文案、简答题流程、图表防遮挡逻辑、图表高度、准确率分母 obj、版本常量 + footer 注入、每日统计结构新增 obj）
- `style.css`（`.sync-row1` 居中、`.sync-note` 样式、`.home-foot` 间距、`.foot .ver`）
- `index.html`（配对弹窗标题、footer 改为版本号占位）
- `CHANGELOG.md`（**新增**，本记录文件）
